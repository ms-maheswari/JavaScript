
## Variables

- Containers for storing data
- It can be declared in **3** ways

## 1. var 
- Cannot reassigned
- Function scope
- Redeclare

```js
var age = 25;
console.log(age);
```

## 2. let
- Reassgined
- Block scope
- Cannot redeclared

```js
let name ="Mahes";
console.log(name);
```

## 3. const
- Cannot reassigned
- Block scope
- Cannot redeclared
```js
const pi = 3.14;
console.log(pi);
```


## 
**Global** - accessible anywhere in the code

**Function** - limited to function where it's declared (var)

**Block** - limited to a block {} (let and const)
